# 📊 Data Visualisation Graphs in Python

A collection of Python programs demonstrating various data visualization techniques using **Matplotlib** and **NumPy**.

This project covers static, dynamic, and animated graph plotting to help understand different visualization methods in Python.

---

## 🚀 Project Overview

This repository includes implementations of:

- 📈 Line Graph (Static & Dynamic)
- 📊 Histogram (Static & Dynamic)
- 🥧 Pie Chart (Static & Dynamic)
- 🔵 Scatter Plot
- 🪜 Stair Plot
- 📑 Subplots
- 🎬 Animated Line Graph
- 🔄 Animated Circle Visualization

The goal of this project is to explore different graph plotting techniques and understand how data can be visually represented using Python.

---

## 🛠 Technologies Used

- Python 3
- Matplotlib
- NumPy

---

## 📂 File Structure

| File Name | Description |
|-----------|-------------|
| `linegraph.py` | Basic line graph |
| `staticline.py` | Static line plot |
| `dynamicline.py` | Dynamically updating line graph |
| `animatedlinegraph.py` | Animated line graph |
| `animatedcircle.py` | Circle animation |
| `histogram.py` | Static histogram |
| `dynamichistogram.py` | Dynamic histogram |
| `piechart.py` | Static pie chart |
| `dynamicpiechart.py` | Dynamic pie chart |
| `scatterplot.py` | Scatter plot |
| `stairplot.py` | Stair-step plot |
| `subplot.py` | Multiple plots in one figure |
